# EZVIZ Patched

A custom component that overrides the built-in EZVIZ integration for testing changes and new features before they are submitted to Home Assistant core.

## Features

- Based on the latest Home Assistant core EZVIZ integration code
- Drop-in replacement: automatically overrides the built-in EZVIZ integration
- Existing configuration continues working seamlessly

## Installation

1. Add this repository as a custom repository in HACS
2. Install "EZVIZ Patched"
3. Restart Home Assistant
4. Your existing EZVIZ configuration will continue working automatically
